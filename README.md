의료서비스의 궁금한 점을 질문하고 본인의 진료받은 기록을 공유하는 사이트입니다. 외국에는 이미 활성화되어있는 의료 커뮤니티 사이트가 많습니다.
사실상 우리나라는 의료보험이 잘되어있어 아플 때 병원에 가면 되지만 정신적 또는 겉으로 표현하기에 민감한 병들은 치료받기 힘들어하는 사람들이 있습니다. 그래서 불편한 환자들을 위해 인식개선과 질병 정보, 치료법, 부작용 등을 공유하는 커뮤니티 사이트를 만들었습니다.

### 1. 프로젝트 설명

Jsp, Servlet을 사용해 개발한 커뮤니티 사이트입니다. 주요 기능은 이메일과 구글 로그인 및 회원가입, 게시판과 댓글, 첨부파일 업로드 등이 있습니다. 

-   자바 웹기반 Servlet과 Jsp를 활용한 웹 어플리케이션 프로젝트
-   기간 : 2020-10-22 ~ 2020-11-17 (3주)
-   팀 인원 : 4명
-   사용기술
    -   OS : windows 10 pro
    -   IDE : Eclipse IDE jee 2020-03, Oracle Sql Developer-19.4.0.354
    -   Front : JSP, CSS, JavaScript, Jquery, Ajax
    -   Back : Java-jdk-8u251, Servlet, Oracle DB, Apache Tomcat,
    -   Tools : Github, GooleDrive, OvenApp.io, Draw.io, Powerpoint 2016
    -   Youtube : [호랭이약방\_Youtube](https://www.youtube.com/playlist?list=PLedGoSru7948Q9sSAPQk1GGt7exbNJm7C)

### 2. 호랭이 약방 - 커뮤니티 사이트

-   기획의도

    - 아이디 기억하기

### 3. 개발 관련 설명

이번 프로젝트에서는 Web 통신과 Java Servlet Container의 기능, 외부 API와 데이터 통신 방식을 학습할 수 있었습니다.
자세한 내용은 다음과 같습니다.
- Uri 주소 입력시 Web Http 통신 방식과 과정에 대해 학습했습니다.
- 정적 콘텐츠와 동적 콘텐츠를 이해하고 Tomcat 웹 서버와 Servlet Container의 특징 및 기능을 학습했습니다.
- 데이터 통신 포맷인 text, image, json를 이해했습니다.
- 비동기 방식인 Ajax 통신을 이해하고, 댓글 기능을 구현했습니다.
- 비회원 권한을 도입하고, 구글 OAuth2 API를 사용해서 사이트 접근성을 높였습니다. 또한 프론트에는 반응형 웹을 구현하여 모바일에서도 접근할 수 있도록 했습니다.
